How to Embed

Paste this code before the closing </body> tag of your website.

Option 1: Same Domain (Direct Backend Host)

<script>
  window.ChatbotConfig = {
    connectionId: "your_connection_id" // e.g., shop_v1
  };
</script>
<script src="http://localhost:5001/widget.js"></script>

Option 2: Cross-Domain (External Website)

If your website is on example.com and backend is on my-api.com, you MUST specify apiUrl.

<script>
  window.ChatbotConfig = {
    connectionId: "your_connection_id",
    apiUrl: "http://localhost:5001" // Your Backend URL
  };
</script>
<script src="http://localhost:5001/widget.js"></script>
